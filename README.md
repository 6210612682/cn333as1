1. สุ่มเลขในช่วง 0 - 1000
2. รับ input จากผู้เล่นแล้วสามารถตอบกลับมาน้อยกว่าหรือมากกว่าเลขที่สุ่มได้
3. หากผู้เล่นทายถูก จะแสดงข้อมูลจำนวนการทายของผู้เล่นและสามารถกด play again เพื่อสุ่มเลขใหม่
